# Monorepo for Swisstronik JS/TS packages

## Overview

This repository contains different packages for interaction with Swisstronik Network

## Packages

| Package                                                          | Description                                                |
|------------------------------------------------------------------|------------------------------------------------------------|
| [@swisstronik/veramo-did-provider](packages/veramo-did-provider) | Plugin for Veramo SDK to issue/update/revoke DID Documents |
| [@swisstronik/sdk](packages/sdk)                                 | SDK for interaction with cosmos-side of Swisstronik Network |
| [@swisstronik/utils](packages/utils)                    | Utils for encryption / decryption of transaction data      |
